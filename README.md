# Craftable Chainmail

A mod for Minecraft beta 1.7.3 allowing one to craft chainmail from
iron rings. Requires [StationAPI](https://github.com/ModificationStation/StationAPI).

## Tags
Makes use of the `c:rings/iron` tag.
